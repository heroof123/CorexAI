================================================================================
                            1. GENEL BAKIŞ
================================================================================

Corex IDE, AI destekli bir kod editörüdür. Tauri 2 framework'ü üzerine inşa
edilmiş olup, React 19 ve Rust kullanarak geliştirilmiştir.

TEMEL ÖZELLİKLER:
- Monaco Editor (VS Code engine) ile kod düzenleme
- Local LLM desteği (LM Studio, Ollama, GGUF)
- Semantic code search (RAG - Retrieval Augmented Generation)
- Git entegrasyonu
- OAuth authentication (GitHub, Microsoft)
- Real-time code analysis

MİMARİ YAKLAŞIM:
3 katmanlı mimari (Presentation, Business, Data)
- Frontend: React 19 + TypeScript
- Backend: Rust + Tauri 2
- Database: IndexedDB + LocalStorage
