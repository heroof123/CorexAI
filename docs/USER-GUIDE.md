## üí° Tips & Tricks

### Productivity

1. **Use Command Palette** - `Ctrl+Shift+P` for quick access
2. **Quick File Open** - `Ctrl+P` to jump to any file
3. **Multi-cursor** - `Alt+Click` for multiple cursors
4. **Split View** - Drag tab to split editor

### AI Best Practices

1. **Be Specific** - Clear questions get better answers
2. **Provide Context** - Mention file names and functions
3. **Review Changes** - Always review AI suggestions
4. **Iterate** - Refine your questions based on responses

### Performance

1. **Close Unused Tabs** - Keep only active files open
2. **Limit File Size** - Large files (>5MB) may be slow
3. **Use Filters** - Filter file tree for faster navigation
4. **Clear Cache** - Restart if performance degrades

## üêõ Troubleshooting

### Common Issues

**AI Not Responding:**
- Check model is loaded
- Verify API key (if using API)
- Check network connection
- Restart Corex

**Files Not Showing:**
- Refresh file tree
- Check .gitignore
- Verify folder permissions
- Re-index project

**Editor Slow:**
- Close large files
- Disable minimap
- Reduce extensions
- Clear cache

**Build Errors:**
- Check terminal output
- Verify dependencies
- Clean and rebuild
- Check file paths
